# Altitude Timesheet - Quick Start Guide

**App URL:** https://altitude-timesheet.pages.dev

---

## For Employees

### Install to Home Screen (Works Like an App)

**iPhone / iPad:**
1. Open Safari, go to https://altitude-timesheet.pages.dev
2. Tap the **Share** button (square with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**

**Android:**
1. Open Chrome, go to https://altitude-timesheet.pages.dev
2. Tap the **3-dot menu** (top right)
3. Tap **Add to Home Screen** or **Install App**
4. Tap **Install**

**Windows Desktop:**
1. Open Chrome or Edge, go to https://altitude-timesheet.pages.dev
2. Click the **install icon** in the address bar (or 3-dot menu > Install App)
3. Click **Install**

### How to Use

1. **Sign In** — Select your name and enter your 4-digit PIN
2. **Fill in each day** — Set start/finish times, select your Job and Task, enter hours per task
3. **Short day?** — If you finish early, a yellow box appears to top up with Sick/Annual/Unpaid Leave
4. **Leave day?** — Change "Working Day" to the leave type (Annual, Sick, RDO, etc.)
5. **Submit** — Tap **Submit Week** when done. Your admin will be notified
6. **My History** — View past weeks, download XLSX, or jump back to any week
7. **Change PIN** — Use the **Change PIN** button at the bottom to update your login PIN

> The app works offline. Your data syncs to the cloud when you're back online.

## For Admin

### Access Admin Panel

Go to: **https://altitude-timesheet.pages.dev/?admin=1234**

(Replace `1234` with your admin PIN if changed)

### Admin Features

| Tab | What it does |
|-----|-------------|
| **Dashboard** | Overview of submitted/pending timesheets for the current week |
| **Employees** | Add, rename, archive employees. View/reset PINs. Set custom hourly rates |
| **Projects** | Create jobs & tasks. Set project type (EBA/Non-EBA/Custom rate). Set budgets & deadlines |
| **Analytics** | Filter by date range, project, employee. View summaries, trends, costs. Export to XLSX |
| **Project Dashboard** | Track budget usage and progress per project |
| **History** | Review all submitted timesheets. Approve, reopen, edit, or delete. Export bulk XLSX |
| **Settings** | Set default hours/times, email recipients, rate tiers, backup/restore, keep-alive |

### Key Admin Tasks

- **Approve timesheets:** History tab > click a submitted card > **Approve**
- **Edit employee data:** History tab > click **Edit** on any week
- **Export reports:** Analytics tab > set filters > **XLSX** button
- **Backup data:** Settings tab > **Download Backup (.json)**
- **Add rate tier:** Settings tab > Hourly Rate Tiers > enter name & rate > **Add Tier**
- **Prevent cloud sleep:** Settings tab > enable **Keep-Alive** (pings every 3 days)
- **Sync to employees:** Projects tab > **Publish to Employees** after adding jobs/tasks

### Tips

- The main page is **view-only** by default. Click **Enable Editing** to modify data manually
- Change your admin PIN in Settings > **Change Admin PIN**
- Use **Bulk Export** in Settings to download all timesheets for a date range
- Use **Cloud Backup** regularly — download the .json file and keep it safe
